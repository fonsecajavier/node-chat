node-chat
========

Installing:

Tested node version: v0.8.7

```sh
npm install
```

In case you still don't have nodemon:
```sh
npm install -g nodemon
```

Install bower components:
```
bower install
```
Start the server by running:
```sh
script/server
```

If you don't want to use nodemon, just start the server by running:
```sh
node app.js
```
